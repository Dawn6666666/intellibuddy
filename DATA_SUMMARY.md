## 📞 技术支持

### 常见问题

**Q1: 数据导入失败？**
- 检查 MongoDB 是否运行
- 检查 `.env` 中的 `MONGO_URI`
- 查看控制台错误信息

**Q2: 前端没有显示数据？**
- 确保后端服务运行中
- 检查浏览器控制台错误
- 清除缓存并刷新

**Q3: 想切换数据集？**
```bash
# 切换到 Java 模块
npm run seed:java

# 切换到默认数据
npm run seed

# 切换回计算机科学课程
npm run seed:cs
```
