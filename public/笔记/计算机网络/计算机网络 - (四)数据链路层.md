### 4.2 ARP协议

```python
def arp_request(ip):
    """ARP请求（简化版）"""
    # 广播：谁是192.168.1.1？告诉我（MAC地址）
    print(f"ARP Request: Who has {ip}? Tell me.")
    
    # 模拟响应
    mac_table = {
        '192.168.1.1': 'AA:BB:CC:DD:EE:FF'
    }
    
    if ip in mac_table:
        print(f"ARP Reply: {ip} is at {mac_table[ip]}")
        return mac_table[ip]
    
    return None

arp_request('192.168.1.1')
```
