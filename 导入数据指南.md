# 📚 导入数据指南（中文版）

## 🎯 最简单的使用方法

### 三步导入数据：

```bash
# 第1步：进入后端目录
cd backend

# 第2步：导入计算机科学课程数据
npm run seed:cs

# 第3步：启动后端服务
npm run dev
```

完成！访问 `http://localhost:5173/app` 查看效果 ✨

## 🔧 常见问题

### Q1: 数据库连接失败？
**A**: 检查 `backend/.env` 文件，确保有：
```env
MONGO_URI=mongodb://localhost:27017/intellibuddy
```

### Q2: 前端没有显示数据？
**A**: 
1. 确保后端服务正在运行 (`npm run dev`)
2. 刷新浏览器页面
3. 检查浏览器控制台是否有错误

### Q3: 想查看所有题目？
**A**: 打开 `QUIZ_PREVIEW.md` 文件，里面有所有86道题的完整预览
